# BioProject

#### Installation with git:

Clone the repository:

```bash
git clone https://github.com/eobw/BioProject.git
```

Move to the folder and install:

```bash
cd BioProject/
python setup.py install
```

#### Check installation

Executing:
```bash
GuessMyLT
```

or

```bash
GuessMyLT -h
```

to display help.
